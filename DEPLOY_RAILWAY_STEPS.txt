🚀 DEPLOY TO RAILWAY - COPY & PASTE GUIDE

═══════════════════════════════════════════════════════════

STEP 1: PUSH TO GITHUB
═══════════════════════════════════════════════════════════

git add .
git commit -m "Deploy to Railway"
git push origin main

(If you don't have a GitHub repo yet, create one first at github.com)

═══════════════════════════════════════════════════════════

STEP 2: GO TO RAILWAY
═══════════════════════════════════════════════════════════

1. Open: https://railway.app
2. Click "Start a New Project"
3. Login with GitHub
4. Click "Deploy from GitHub repo"
5. Select: tds-project-1 (or your repo name)

═══════════════════════════════════════════════════════════

STEP 3: ADD ENVIRONMENT VARIABLES
═══════════════════════════════════════════════════════════

Click on your service → Variables tab → Add these one by one:

GITHUB_TOKEN
your_github_token_here

GITHUB_USERNAME
subhuchan

GEMINI_API_KEY
your_gemini_api_key_here

USER_SECRET
your_secret_here

DATABASE_URL
sqlite:///./tds_projects.db

═══════════════════════════════════════════════════════════

STEP 4: WAIT FOR DEPLOYMENT
═══════════════════════════════════════════════════════════

Railway will automatically:
- Detect it's a Python + Node.js project
- Install dependencies
- Build frontend
- Start backend
- Give you a URL

Wait 5-10 minutes for first deployment.

═══════════════════════════════════════════════════════════

STEP 5: GET YOUR URL
═══════════════════════════════════════════════════════════

1. Click "Settings" tab
2. Scroll to "Domains"
3. Click "Generate Domain"
4. Copy your URL: https://your-app.up.railway.app

═══════════════════════════════════════════════════════════

STEP 6: TEST YOUR APP
═══════════════════════════════════════════════════════════

1. Open your Railway URL
2. You should see the dashboard
3. Click "Builder"
4. Create a test project
5. Check if it works!

═══════════════════════════════════════════════════════════

✅ DONE! YOUR APP IS LIVE! 🎉

═══════════════════════════════════════════════════════════

TROUBLESHOOTING:

If deployment fails:
1. Check "Deployments" tab for logs
2. Verify all environment variables are set
3. Make sure your GitHub repo is up to date

If app doesn't load:
1. Check if domain is generated
2. Wait a few more minutes
3. Check deployment logs for errors
